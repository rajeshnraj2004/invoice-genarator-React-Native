# 🚀 Invora App

Invora is a modern **React Native mobile application** designed for managing invoices, user profiles, and essential app features with a clean and intuitive UI.

---

## 📱 Features

* 🔐 User Authentication (Local Storage / AsyncStorage)
* 👤 Profile Section with Avatar (Initial-based)
* 🧾 Invoice Management (Create & View)
* ⚙️ Settings Screen
* ➕ Floating Action Button (FAB) for quick actions
* 🎨 Modern iOS-style UI with Bottom Navigation
* ⚡ Smooth Animations using React Native Animated API

---

## 🛠️ Tech Stack

* **React Native**
* **Expo**
* **AsyncStorage**
* **React Navigation**
* **Vector Icons (Ionicons)**

## ⚙️ Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/invora-app.git
```

2. Navigate to project:

```bash
cd invora-app
```

3. Install dependencies:

```bash
npm install
```

4. Run the app:

```bash
npx expo start
```

---

## 🚀 Usage

* Open the app on emulator or mobile
* Navigate using bottom tab bar
* Use ➕ button to add new items (invoices, data, etc.)
* Manage profile and settings
